IoT Projects

Projects focused on IoT and distributed sensor and data collection.

ProjectCategoryDescriptionPrime Moover NotesWebsite
HeliumConnectivityDecentralized wireless network for IoT devices. Backed by Andreessen Horowitz. HNT token. Over 1M hotspots, expanding to 5G.Heavily invested as node operator since inception. Mature, unique service. Mobile data backhaul great add. IoT adoption slower than expected.helium.com
NodleConnectivityDecentralized IoT network via Bluetooth using smartphones as nodes. NODL token.IoT uplink like AirTags. Not comfortable with battery impact and security exposure for mobile device.nodle.io
ChirpConnectivityDecentralized wireless infrastructure on Sui blockchain. Requires Blackbird gateway. CHIRP token.Interesting project to map wireless signals. Obfuscated purchasing system prevents investment.chirptoken.io
WeatherXMData CollectionGlobal weather sensor network. Revenue from station sales and data licensing. $WXM token. 5,000+ stations.Early adopter. Clear market for data. Difficult to scale for most individuals.weatherxm.com
DIMOAutomotive IoTDecentralized vehicle IoT data monetization. Raised $9M seed. DIMO token.Clear purpose and market. Not for me—don't drive much.dimo.zone
Peaq NetworkBlockchain InfraDecentralized infrastructure for Machine Real-World Assets (RWA). PEAQ token.Potential for unified interface and IoT data aggregation. Not clear how to participate.peaq.network
IoTeXIoT BlockchainBlockchain connecting IoT devices securely. IOTX token. Funded by top blockchain VCs.Clear purpose and revenue model. Delegator requires 1.2M IOTX—out of budget.iotex.io
MeshtasticMesh NetworkingOpen-source mesh network for long-range IoT using radio devices. Non-monetized.Interesting DIY project. No money in it but great for off-grid texting.meshtastic.org
FreedomFiCellularDecentralized open 5G network. Requires specialized hardware. Tied to HNT.Good idea, some barrier to entry. Would invest if in dense/transient areas.freedomfi.com
StreamrData StreamingDecentralized real-time data streaming. DATA token.Singular focus and clear target market. May revisit.streamr.network
HivemapperMappingDecentralized mapping via dashcams. HONEY token.Interesting but growing competition. Unclear special sauce.hivemapper.com
WicryptConnectivityDecentralized mobile internet sharing. WNT token.Many last-mile wifi projects exist commercially. Not for me.wicrypt.com
Air Quality EggEnvironmentalCommunity-led air quality monitoring. Non-profit, open data.Emerging project. Challenged by who uses data and sustainability. Not for me.airqualityegg.com
